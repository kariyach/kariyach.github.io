Ⅲ. チャンクオプション

1. チャンクごとに設定

  • チャンクを実行しない
    • 一部のチャンクだけ動かないなどの理由で､レンダリングやすべてのチャンクの実行から除外したい場合に有効

- 図にタイトルを付ける - 図の比率を変える - コードを折り畳む

2. ファイル全体に設定

YAMLに以下を追加すると､すべてのチャンクでエラーを非表示､コードは折り畳んで表示される（挿入場所はどこでもよい）

execute:
  error: false
code-fold: true

スペースに注意（必ず半角スペースで空きスペースを作ること）

Ⅳ. CSSと出力テンプレート

0. 事前準備

  • Google Drive「CSSとテンプレート」から以下のファイルをダウンロード
    1. CSSファイル（「style.css」）
    2. テンプレートファイル（「template.pptx」）

1. html

  1. Quartoファイルと同一階層に､CSSファイルを置く

  2. YAMLに以下のコードを追加（場所は任意）

css: style.css

htmlカラーコードで色を変えられる。カラーコードは色の名前とカラーコードが一目でわかるWEB色見本を参照

2. Power Point

  1. Quartoファイルと同一階層に､テンプレートファイルを置く
  2. YAMLに､html出力をコメントアウトしたうえで､以下を追加
format:
  pptx:
    toc: true
    reference-doc: template.pptx
  • pptxで可能な設定については公式サイトを参照

テンプレートファイルの作成

このサイト（RStudioとRMarkdownで作るPowerPointプレゼンテーション）が参考になります

テンプレートに指定するファイルは、次の4つのレイアウトを最初の4つのレイアウトとして含む必要があります。

    1. タイトルスライド（Title Slide）
    1. セクション見出し（Section Header）
    1. タイトルとコンテンツ（Title and Content）
    1. 2つのコンテンツ（Two Content）

最近のバージョンのPowerPointに含まれているテンプレート、あるいはそれを編集したテンプレートであれば、おそらく問題なく使えるでしょう。

Ⅴ. テキストラベル

penguins %>%
  group_by(species) %>% 
  summarise( 
    個数 = n()
  ) %>%
  ggplot(aes(x = species, y = 個数)) +
  geom_col() +
  geom_text(aes(label = 個数), vjust = -0.5,
            size = 4) # labelでラベルを指示（オブジェクトを指定）。vjust でラベルの位置を調整
penguins %>%
  group_by(species) %>% 
  summarise( 
    個数 = n()
  ) %>%
  mutate(
    比率 = round(個数/sum(個数) *100, 1)
  ) %>% 
  ggplot(aes(x = species, y = 比率)) +
  geom_col() +
  geom_text(aes(label = 比率), vjust = -0.5,
            size = 4) # labelでラベルを指示（オブジェクトを指定）。vjust でラベルの位置を調整
penguins %>%
  group_by(species) %>% 
  summarise( 
    個数 = n()
  ) %>%
  mutate(
    比率 = round(個数/sum(個数) *100, 1)
  ) %>% 
  ggplot(aes(x = species, y = 比率)) +
  geom_col() +
  geom_text(aes(label = paste0(比率, "%")), vjust = -0.5,
            size = 4) # 数値に%を付ける
