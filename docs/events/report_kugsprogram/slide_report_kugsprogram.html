Ⅲ. 先行研究

本庄秀明「大学生の学習態度とその規定要因について：全国大学生調査の二次分析の結果から」

  • 対象
    • 「全国大学生調査」（2007年に東京大学大学院教育学研究科大学経営・政策センターが実施）
  • 目的
    1. 学習態度（主体的学習者と受動的学習者）による学習行動と学習成果の獲得状況の違い
    2. 学習態度の規定要因
  • 知見
    1. 主体的学習態度がプラスの影響を及ぼす
    2. 授業方法よりも授業内容自体が重要である

    「自己認識の確立に役立つ授業内容は主体的学習態度に結びつきやすい」(本庄 (2022), 99)

井ノ上憲司ほか「多面的・総合的入学者選抜の効果検証：卒業年度の学生・教員双方の追跡調査の分析」

  • 目的

AO・推薦入試では，学問・研究に意欲を持ち，主体性をもって周囲と協働しながら学べる学生を本当に獲得できているのか (井ノ上 ほか (2022), 60)

  • 目的と方法
    1. 入学時の学生の学問・研究に対する意欲に関する意識調査
    2. 4年間のGPAの推移
    3. 学問・研究に対する意欲主体性等の評価にかかわる意識調査
    4. 卒業論文の指導教員を対象とする学生評価
  • 知見

「多面的・総合的な評価による選抜は，手間はかかるが望ましい学生を獲得する手段として有効」と結論づける (井ノ上 ほか (2022), 61)
